# Toast

Toasts are used to show brief, non-intrusive feedback messages to the user. They appear at the bottom of the Vicinae window and disappear automatically.

## showToast

```tsx
import { showToast, Toast } from '@vicinae/api';

await showToast({ title: 'Success', style: Toast.Style.Success });
```
