# Actions

The `Action` component represents an individual action within an `ActionPanel`. Vicinae provides several built-in action types for common operations.

## Action

The base `Action` component for custom actions.

```tsx
import { Action } from '@vicinae/api';

<Action title="Say Hello" onAction={() => console.log('Hello!')} />
```
