# OAuth

The OAuth API provides utilities for implementing OAuth 2.0 authorization flows in extensions. This is useful for connecting to third-party services that require user authentication.

## Example

```tsx
import { OAuth } from '@vicinae/api';

const client = new OAuth.PKCEClient({
  redirectMethod: OAuth.RedirectMethod.Web,
  providerName: 'GitHub',
  providerIcon: 'github.png',
  description: 'Connect your GitHub account',
});
```
