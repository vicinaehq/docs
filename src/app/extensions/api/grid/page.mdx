# Grid

The `Grid` component is used to display items in a grid layout. Useful for visual content like icons, images, or cards.

## Example

```tsx
import { Grid } from '@vicinae/api';

export default function Command() {
  return (
    <Grid>
      <Grid.Item content={{ value: "icon.png" }} title="Item 1" />
    </Grid>
  );
}
```
