# Colors

The `Color` module provides a set of semantic colors for use in components like tags, icons, and backgrounds.

## Usage

```tsx
import { Color, List } from '@vicinae/api';

<List.Item title="Important" icon={{ source: Icon.Circle, tintColor: Color.Red }} />
```
