# List

The `List` component is used to display a vertical list of items with optional search, sections, and detail views.

## Example

```tsx
import { List } from '@vicinae/api';

export default function Command() {
  return (
    <List>
      <List.Item title="Item 1" />
      <List.Item title="Item 2" />
    </List>
  );
}
```
