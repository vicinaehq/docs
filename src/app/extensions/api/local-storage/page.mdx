# Local Storage

Persistent key-value storage for extensions. Data persists across command runs and is not automatically evicted, unlike `Cache`.

## Store and Retrieve

```tsx
import { LocalStorage } from "@vicinae/api";

await LocalStorage.setItem("favoriteFruit", "🍎");

const fruit = await LocalStorage.getItem<string>("favoriteFruit");
```

## Remove

```tsx
import { LocalStorage } from "@vicinae/api";

await LocalStorage.removeItem("favoriteFruit");
```

## List All Items

```tsx
import { LocalStorage } from "@vicinae/api";

const items = await LocalStorage.allItems();
// { favoriteFruit: "🍎", count: 42 }
```

## Clear

Remove all stored items for this extension.

```tsx
import { LocalStorage } from "@vicinae/api";

await LocalStorage.clear();
```

## Supported Value Types

Values can be `string`, `number`, or `boolean`. For complex data, serialize with `JSON.stringify` and parse back with `JSON.parse`.
