# Local Storage

The Local Storage API provides persistent key-value storage for extensions. Unlike `Cache`, data stored here is meant to be long-lived and is not automatically evicted.

## Functions

### LocalStorage.getItem

```tsx
import { LocalStorage } from '@vicinae/api';

const value = await LocalStorage.getItem<string>('key');
```
