# Images

Types for specifying images across the extension API.

## ImageLike

Anywhere Vicinae expects an image, you can pass an `Icon`, a URL, an asset path, or a full `Image` object.

```tsx
import { Icon, List } from "@vicinae/api";

<List.Item title="From Icon" icon={Icon.Star} />
<List.Item title="From Emoji" icon="🍎" />
<List.Item title="From Asset" icon="apple.png" />
```

## Image Object

For more control, use the full `Image` type with tint and mask options.

```tsx
import { Color, Icon, Image, List } from "@vicinae/api";

<List.Item
  title="Tinted"
  icon={{ source: Icon.Circle, tintColor: Color.Green }}
/>
<List.Item
  title="Masked"
  icon={{ source: "avatar.png", mask: Image.Mask.Circle }}
/>
```

## Themed Images

Provide different images for light and dark themes.

```tsx
const icon = { light: "icon-light.png", dark: "icon-dark.png" };
```
