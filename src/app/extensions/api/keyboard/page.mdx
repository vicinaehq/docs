# Keyboard

The Keyboard module provides types and utilities for working with keyboard shortcuts in actions and commands.

## Keyboard.Shortcut

```tsx
import { Action, Keyboard } from '@vicinae/api';

<Action
  title="Refresh"
  shortcut={{ modifiers: [Keyboard.Modifier.Cmd], key: "r" }}
  onAction={() => {}}
/>
```
