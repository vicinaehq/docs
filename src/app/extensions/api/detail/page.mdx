# Detail

The `Detail` component is used to present markdown content with an optional metadata sidebar. It is typically used as a root view or within a `List.Item.Detail`.

## Example

```tsx
import { Detail } from '@vicinae/api';

export default function Command() {
  return <Detail markdown="# Hello World" />;
}
```
