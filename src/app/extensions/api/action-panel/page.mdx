# ActionPanel

The `ActionPanel` component provides a list of actions that the user can perform in the current context. It is typically placed inside a `List.Item`, `Grid.Item`, `Detail`, or `Form`.

## Example

```tsx
import { List, ActionPanel, Action } from '@vicinae/api';

export default function Command() {
  return (
    <List>
      <List.Item
        title="Item"
        actions={
          <ActionPanel>
            <Action title="Do something" onAction={() => console.log('done')} />
          </ActionPanel>
        }
      />
    </List>
  );
}
```
