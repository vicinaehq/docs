# Operating System Support

Vicinae will eventually support all major operating systems. Linux is the primary development target, and macOS support is in beta:

| Name | Supported | Comment |
|------|-----------|---------|
| Linux Wayland | ✅ | Primary development target, recommended |
| Linux X11 | ✅ | Generally less polished than its Wayland counterpart |
| macOS | ✅ | Beta since v0.23.0, Apple Silicon only |
| Windows | ❌ | Planned, no ETA |
| Mobile things | ❌ | Not planned |
